# Mechanical Engineering Portfolio

A polished static portfolio for GitHub Pages.

## What to edit

- Replace `you@example.com` in `index.html` with your real email.
- Swap the placeholder project names and descriptions with your actual work.
- Add your resume link if you have one.
- If you want a custom domain, add a `CNAME` file with your domain once GitHub Pages is configured.

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Push these files to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Choose the branch and root folder.
5. Add your custom domain there if you want to use the student-pack domain.
