IMS bearing failure prediction

A machine learning side project using public IMS bearing vibration data to predict failures. The dataset comes from test-to-failure experiments with 1-second vibration snapshots recorded at 20 kHz.

  • 3 data sets and 20,480-point samples
  • Feature extraction from raw vibration signals
  • Outer race and roller defect prediction

Data Science / ML · Part 2

Universal bearing RUL — cross-dataset study

Does a prognostics model transfer to a rig it has never seen? Three public run-to-failure datasets (IMS, XJTU-SY, FEMTO — 36 bearings, lifetimes from 37 minutes to 45 days) pooled into one life-fraction model, with the generalization gap measured directly.

  • Geometry-derived envelope features, physics-validated
  • Leave-one-bearing-out vs leave-one-dataset-out
  • Honest reporting: worst cases and leakage checks included

Data Engineering / Live Pipeline

IST flight delay dataset

There is no public historical delay dataset for Istanbul Airport — so I'm building one. A zero-maintenance GitHub Actions pipeline collects every IST flight daily, joins actual movements with schedules, and archives METAR weather alongside.

  • OpenSky + aviationstack joined on callsign
  • Append-only CSVs with quality flags
  • Phase 2: a delay prediction model
