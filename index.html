Data Science / ML

IMS bearing failure prediction

A machine learning side project using public IMS bearing vibration data to predict failures. The dataset comes from test-to-failure experiments with 1-second vibration snapshots recorded at 20 kHz.

  • 3 data sets and 20,480-point samples
  • Feature extraction from raw vibration signals
  • Outer race and roller defect prediction

Web / Live Data

Live Turkish Airlines flight globe

A real-time web app that plots every airborne Turkish Airlines flight on an interactive 3D globe, using live ADS-B transponder data from the OpenSky Network.

  • WebGL globe with 130+ flights tracked live
  • Aircraft size classes and cargo vs passenger split
  • 15-second polling, search, camera focus on click
